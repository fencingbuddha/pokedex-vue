# Vue Pokédex

Small Pokédex built with **Vue 3 + TypeScript + Vite**, using the public [PokéAPI](https://pokeapi.co/).

### Features

- Search Pokémon by name or ID (e.g. `pikachu`, `1`, `25`)
- Shows official sprite, ID, and name
- Basic loading and error states

### Running locally

```bash
npm install
npm run dev
